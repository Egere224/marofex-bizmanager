# Marofex BizManager

A full-stack business management system for inventory tracking, sales management, and customer debt tracking.

## 🚀 Features

- Inventory management
- Sales tracking
- Customer debt tracking
- Dashboard analytics

## 🛠️ Tech Stack

- Frontend: React + Vite + Tailwind
- Backend: Node.js + Express
- Database: PostgreSQL

## 📦 Project Structure

- /client → frontend
- /server → backend
- /database → SQL schema

## ⚙️ Setup

1. Clone the repo
2. Install dependencies in client and server
3. Add `.env` file
4. Run the app
